# Divider 分割线

<p>分隔内容的文档分割线</p>

#### 何时使用

- 对不同章节的文本段落进行分割。

- 对行内文字/链接进行分割，例如表格的操作列。

## API

| Name        | Description | Type                    | Default  |
| ----------- | ----------- | ----------------------- | -------- |
| style       | 自定义样式  | `CSSProperties`         | `--`     |
| className   | 类名        | `string`                | `--`     |
| fontSize    | 字体大小    | `string`                | `14px`   |
| borderColor | 字体颜色    | `string`                | `--`     |
| align       | 对齐方式    | `left / right / center` | `center` |
